# SmartCity AI

## Команда
Key2Life

## Описание проекта

SmartCity AI — интеллектуальная система автоматической обработки обращений граждан.

Система анализирует текст обращения, определяет категорию проблемы и уровень её приоритетности. Это позволяет городским службам быстрее реагировать на обращения жителей и эффективнее распределять ресурсы.

## Основные возможности

- Автоматическая классификация обращений
- Определение приоритета заявки
- Быстрая обработка текстовых запросов
- Удобный интерфейс для демонстрации

## Категории обращений

- Дороги
- Освещение
- Мусор
- Безопасность
- Транспорт

## Используемые технологии

- Python
- Scikit-learn
- Pandas
- Gradio

## Практическая ценность

Проект может использоваться городскими службами для автоматизации обработки обращений и повышения качества городских сервисов.

## Будущее развитие

- Мобильное приложение
- Интеграция с городскими порталами
- Карта обращений жителей
- Аналитика проблемных районов города
