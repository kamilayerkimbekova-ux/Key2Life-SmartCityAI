# SmartCity AI — Key2Life 🏙️🤖

Интеллектуальная система автоматического анализа, категоризации и приоритизации обращений жителей города Алматы, разработанная в рамках хакатона **SmartScape**. Проект включает в себя ML-бэкэнд на Flask и интерактивный веб-интерфейс с картой на Leaflet.js.

## 🚀 Описание проекта

**SmartCity AI** автоматизирует обработку жалоб граждан, используя алгоритмы машинного обучения. Система мгновенно классифицирует входящий текст по городским категориям и определяет критичность (приоритет) проблемы, отображая статистику и геометку на карте в реальном времени. Это позволяет городским службам реагировать на критические инциденты в разы быстрее.

### Ключевой функционал:
* **Интеллектуальный анализ текста:** Определение категории проблемы (Дороги, Освещение, Мусор, Безопасность, Транспорт).
* **Динамическая приоритизация:** Оценка критичности заявки (Высокий или Средний приоритет) на основе лексического анализа текста.
* **Интерактивная карта:** Визуализация обращений на карте города Алматы (Leaflet.js) с цветовой кодировкой маркеров в зависимости от категории.
* **Статистика в реальном времени:** Наглядные счетчики и дашборд обработки заявок по секторам.

## 🛠️ Технологический стек

* **Backend:** Python, Flask, Flask-CORS
* **Machine Learning:** Scikit-learn (TF-IDF Vectorizer + Logistic Regression)
* **Data Processing:** Pandas, Numpy
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla), Leaflet.js

## ⚙️ Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/your-username/Key2Life-SmartCityAI.git
cd Key2Life-SmartCityAI
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

Активировать окружение:

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск приложения

```bash
python smartcityai.py
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:5050
```

## 🤖 Как работает система

1. Пользователь вводит описание городской проблемы.
2. Алгоритм TF-IDF преобразует текст в числовые признаки.
3. Модель Logistic Regression определяет категорию обращения.
4. Вторая модель оценивает уровень приоритета.
5. Результат отображается на веб-интерфейсе вместе со статистикой и геометкой на карте.

## 📡 API

### Анализ обращения

**POST** `/api/analyze`

Пример запроса:

```json
{
  "text": "Большая яма на дороге возле школы"
}
```

Пример ответа:

```json
{
  "category": "Дороги",
  "priority": "Высокий",
  "cat_confidence": 98.2,
  "pri_confidence": 95.7
}
```

### Получение статистики

**GET** `/api/stats`

Возвращает текущую статистику обработанных обращений.
